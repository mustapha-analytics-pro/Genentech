Vendor-side issues found (separate from pipeline)

  • Activity SFTP drop is missing Apr 28 entirely. Vendor's dcm_20260429 weekly drop has only 6 days of activity data (Apr 22–27). Glue job ran before Apr 28 daily activity file landed.
  • Clicks Apr 24 contains 1,734 duplicate rows on SFTP. Raw count 257,257 minus unique-set count 255,523 = 1,734 dup rows. Vendor-side Spark output emitted some Apr 24 events twice. Local has no duplicates.
  • Rich_media Apr 28 emitted TWICE on SFTP. Two run timestamps (run-1777449796547 and run-1777450124229) both contain Apr 28 data with byte-identical sample parts. Vendor's drop double-counts Apr 28 by ~46M rows.
